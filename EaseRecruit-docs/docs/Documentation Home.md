# Introduction

Welcome to the **EaseRecruit Documentation**! This project is designed to streamline the recruitment process by providing a comprehensive platform for both job seekers and employers.

## Overview

EaseRecruit offers a wide range of features to simplify and enhance the recruitment process. It is built using **React** for the frontend, ensuring a responsive and dynamic user experience. The platform leverages **Cloudinary** for efficient image and video management, allowing users to upload and manage media seamlessly.

On the backend, we utilize **Node.js** and **Express.js** to create a robust and scalable server, handling requests and data efficiently for smooth application performance.

### Key Features

#### **For Job Seekers**:

- **Signup & Login**: A user-friendly authentication system with email notifications for account creation and password resets.
- **Apply for Jobs**: Easily browse job listings and submit applications.
- **Favorite Jobs**: Save jobs you're interested in for easy access later.
- **Real-time Notifications**: Get instant updates on job applications, interview invitations, and other activities.
- **Inbox for Chat**: Direct messaging system for communication between job seekers and employers.
- **Dashboard**: A personal dashboard featuring real-time charts and statistics, including profile views and job application statuses.

#### **For Employers**:

- **Post and Manage Jobs**: Employers can post, edit, and manage job listings with ease.
- **Profile Management**: Update and manage company profiles.
- **Candidate Management**: View the list of applicants for each job posting and manage the recruitment process.
- **Dashboard**: A comprehensive dashboard displaying job statistics, application trends, and profile views.

### Technologies Used

- **Frontend:**

  - **React**: A JavaScript library for building user interfaces.
  - **Cloudinary**: A cloud-based service for managing images and videos.
  - **Axios**: For API requests.
  - **Redux Toolkit**: State management for a smooth and efficient application experience.
  - **Recharts**: For rendering real-time charts on the dashboards.
  - **React Router**: For managing routes within the application.
  - **React Hook Form** and **Yup**: For form handling and validation.
  - **GSAP**: For smooth animations and interactive features.

- **Backend:**
  - **Node.js**: A JavaScript runtime built on Chrome's V8 JavaScript engine.
  - **Express.js**: A web application framework for building APIs and handling server-side logic.
  - **Mongoose**: For interacting with MongoDB and managing data.
  - **JWT**: For secure user authentication.
  - **Nodemailer**: For sending emails related to job applications, account management, and notifications.
  - **Cloudinary**: For media storage and optimization.
  - **Bcrypt**: For securely hashing user passwords.
  - **Winston**: For logging and monitoring backend operations.

## Getting Started

To begin using EaseRecruit, set up your development environment by following the steps below.

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) version 18.0 or above:
  - When installing Node.js, ensure that you check all checkboxes related to dependencies.

## Installation

Clone the repository and install the necessary dependencies:

```bash
git clone
cd easerecruit
npm install
```
